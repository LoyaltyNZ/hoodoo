Purpose

Structured logging onto an AMQP-based queue, via the AMQEndpoint gem. Optional; class is defined only if the supporting AMQEndpoint gem's classes are defined.

The middleware uses this to put log and error messages on the queue. Interested services use this to read such messages from the queue.
