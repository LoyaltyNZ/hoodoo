Purpose

A subclass of Hoodoo::Services::Middleware in order to gain access to middleware protected methods, this code is used for inter-resource calls. Instances of this class are given to service action implementations through the context variable, through Hoodoo::Services::Context#resource.

This class is almost a private implementation detail of Hoodoo::Services::Middleware and is namespaced inside it. File “service_middleware.rb” must be “require”'d first.

11-Nov-2014 (ADH): Split out from service_middleware.rb.
