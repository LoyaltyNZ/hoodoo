Module used as a namespace to collect classes that represent Resources documented by the your platform's API. Each is an Hoodoo::Presenters::Base subclass, so can be used to render and validate JSON data.

Hoodoo::Data::Resources::Version is defined as an example class. The Hoodoo::Data::Resources::* namespace is otherwise free for use by Hoodoo clients (indeed, types or resources that wish to reference one another through the DSL must use this namespace).
