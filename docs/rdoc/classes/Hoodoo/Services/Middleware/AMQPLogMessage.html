For AlchemyAMQ gem users, the AMQPLogMessage class provides an AlchemyAMQ::Message subclass used for sending structured log data to the queue. Hoodoo::Logger uses this.

See the AlchemyAMQ gem for more details.

reported_at=( time ) Link

Set the time read back by reported_at using a Time instance. This is formatted internally as a String via TIME_FORMATTER and reported as such in subsequent calls to reported_at.

Conversion from Time to String is done here, rather than by the caller setting this instance's variables, so that we can internally enforce the accuracy required for this field.

time

The Time instance to set (and process into a string internally via TIME_FORMATTER), or a String instance already so formatted, or nil to clear the value.

Source: show 

# File lib/hoodoo/services/middleware/amqp_log_message.rb, line 91
def reported_at=( time )
  if time.is_a?( String )
    @reported_at = time
  elsif time.is_a?( Time )
    @reported_at = time.strftime( TIME_FORMATTER )
  else
    @reported_at = nil
  end
end
