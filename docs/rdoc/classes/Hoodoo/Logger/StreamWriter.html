new( output_stream = $stdout ) Link

Create a stream writer instance. Although you could initialize this class with a slow output stream, they're expected to be fast (e.g. terminal output) as this is an Hoodoo::Logger::FastWriter subclass.

For reliable file writing, use the Hoodoo::Logger::FileWriter class instead.

output_stream

Optional I/O stream instance; default is +$stdout+. The instance must provide a puts implementation.
