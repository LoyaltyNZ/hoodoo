Module Hoodoo::Monkey::Patch::NewRelicTracedAMQP

  • lib/hoodoo/monkey/patch/newrelic_traced_amqp.rb

Wrap Hoodoo::Client::Endpoint::AMQP using NewRelic transaction tracing so that over-queue inter-resource calls get connected together in NewRelic's view of the world.

This module self-registers with Hooodoo::Monkey and, provided that Hoodoo::Services::Middleware is defined at parse-time and Hoodoo::Services::Middleware.environment.production? reports false, will be enabled by default.
