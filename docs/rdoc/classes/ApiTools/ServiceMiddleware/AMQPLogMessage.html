Class ApiTools::ServiceMiddleware::AMQPLogMessage < AMQEndpoint::Message

  • lib/api_tools/service_middleware/amqp_log_message.rb

For AMQEndpoint gem users, the AMQPLogMessage class provides an AMQEndpoint::Message subclass used for sending structured log data to the queue. ApiTools::ServiceMiddleware::StructuredLogger uses this.

See the AMQEndpoint gem for more details.
