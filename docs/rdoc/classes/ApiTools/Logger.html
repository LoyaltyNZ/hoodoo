Class ApiTools::Logger < Object

  • lib/api_tools/logger.rb

A class intended as a standardised API/service logger. The class acts as a proxy; an external logger can (and usually will) be set up through ::logger=. If a logger is not set, the class will log to STDOUT with an appropriate 'DEBUG', 'INFO', 'WARN' or 'ERROR' prefix.

Custom loggers are written by implementing the same interface as the four methods ::debug, ::info, ::warn and ::error described herein. Loggers just take the list of arguments and log them. There is no need to check log level as ApiTools only calls a custom logger if the appropriate level is set.

level=(level) Link

Set the current log level.

level

One of :debug, :info, :warn, :error. These are in order from most verbose to least. When :debug, all kinds of log messages will be written. When :info, all kinds of log messages except debug will be written. When :warn, all messages except debug or info will be written; when :error, only errors will be written.
